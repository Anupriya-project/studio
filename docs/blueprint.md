# **App Name**: PyMentor

## Core Features:

- Content Management System (Admin): Secure administrative interface to upload, edit, and delete all learning content, including programs, notes, PDFs, and video links.
- Python Programs Display: Present a variety of tested Python programs with source code, correct output, common error explanations, and clear instructions on how to run them.
- Notes & PDF Explanations: Display easy-to-understand explanations of Python concepts and allow users to download associated PDF reference files.
- Embedded Tutorial Videos: Show embedded YouTube tutorial videos categorized by difficulty level or topic.
- Intelligent Search & Summarization Tool: A powerful search system that provides relevant answers from the learning platform's content, utilizing an AI tool to summarize search results for quick comprehension.
- Home Page & Learning Roadmap: A clear, engaging home page featuring an introduction to the platform and a structured, beginner-to-advanced learning roadmap for Python.
- Responsive Navigation: A fully visible, clickable, and responsive navigation menu accessible across all pages, ensuring a seamless user experience on any device.

## Style Guidelines:

- A deep, focused blue-violet (#6D26D9) serving as the primary color, suggesting intelligence and modern technology while providing a calming effect for concentration.
- A clean, off-white background color (#F5F3F7) with a subtle blue-violet hue, providing an optimal reading canvas and promoting a sense of clarity.
- A vibrant, bright blue (#4D66F0) used as an accent, designed to highlight interactive elements, calls to action, and key information.
- Body and headline font: 'Inter', a modern sans-serif that ensures excellent readability for technical and educational content across all screen sizes due to its clean, objective appearance.
- Use minimalist, clear, and modern educational and programming-related icons to enhance user understanding without cluttering the interface.
- Implement a clean, organized, and responsive grid layout across all pages to ensure content readability and easy navigation on desktop and mobile devices.
- Subtle, purposeful animations for page transitions and interactive elements to provide visual feedback and a polished user experience without distractions.
